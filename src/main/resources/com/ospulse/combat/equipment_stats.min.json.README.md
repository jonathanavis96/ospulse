# equipment_stats.min.json — provenance & regeneration

A compact `{ "<itemId>": [stats...] }` map of equipment combat bonuses,
**derived directly from the OSRS game cache** (item `params`) — raw facts,
not a wiki compilation, so free of the CC-BY-NC-SA NonCommercial block.

## Array order (16 ints)
```
[astab, aslash, acrush, amagic, arange, dstab, dslash, dcrush, dmagic, drange, str, rstr, mdmg, prayer, speed, slotId]
```
- `mdmg` = magic damage, tenths-of-percent as the cache stores it (150 = +15%).
- `speed` = attack speed in ticks; **0 means derive from weapon category**
  (staves / powered weapons carry no per-item speed param).
- `slotId` = RuneLite EquipmentInventorySlot id where the cache exposes it
  (3=weapon, 0=head, 13=ammo, ...); **-1 = not cache-exposed** (best-effort).
- `is2h` is intentionally NOT included: no cache param reliably encodes it
  (render/wear data, not a stat). The plugin resolves 2h at runtime from
  RuneLite's ItemEquipmentStats.

## Oracle (validation only — NOT shipped)
The param ids above were confirmed by matching cache values against
weirdgloop `osrs-dps-calc` `equipment.json` across the item overlap. That data
is used solely to validate/calibrate; the shipped numbers are cache-derived.
