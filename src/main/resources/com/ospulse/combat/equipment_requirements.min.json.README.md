# equipment_requirements.min.json — provenance & regeneration

A compact `{ "<itemId>": { "<skillName>": <level>, ... } }` map of item
skill/level equip requirements, **derived directly from the OSRS game cache**
(item `params`) — raw facts, not a wiki compilation, so free of the CC-BY-NC-SA
NonCommercial block.

## Schema
```
{ "<itemId>": { "<skillName>": <level>, ... }, ... }
```
An item appears only if it carries at least one requirement. Most items have
one; a few dual-requirement items (e.g. Ancestral hat) have two.

## Source
- Cache: OpenRS2 archive id **2615**, build **231**.
